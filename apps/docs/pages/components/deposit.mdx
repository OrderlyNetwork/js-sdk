import { LiveProvider, LiveEditor, LivePreview, LiveError } from "react-live";
import {Deposit,AssetsProvider,Page,Button,useModal,modal} from '@orderly.network/react'

# Deposit component

Block component allowing users to deposit.

## Features

- The component is already integrated with `@orderly.network/hooks` and can be used without any configuration.
- The component handles checks on the user's status, as well as connection to the wallet.
- The component must be used within the `ConnectionProvider` component within `OrderlyAppProvider`.

## Usage

### Static mode

You can display the `Deposit` component directly in your project:

<LiveProvider code={`<Page>
    <Deposit/>
</Page>`} scope={{AssetsProvider,Deposit,Page}}>
<div className="flex justify-center py-3 mt-5 rounded shadow-lg bg-base-700">

<LivePreview />

</div>
  
  <LiveError />
</LiveProvider>

```typescript copy
import { Deposit } from "@orderly.network/react";

export const App = () => {
  return (
    <Deposit />
  );
};
```

### Modal mode

You can also choose to displays the `deposit` component through a modal:

<LiveProvider code={`()=>{
    return (
        <Page>
        <Button onClick={()=>{
            dialog({
                title:'Deposit',
                content:<Deposit/>
            });
        }}>Open deposit dialog</Button>
</Page>
    )
}`} scope={{AssetsProvider,Deposit,Page,Button,dialog:modal.dialog}}>
<div className="flex justify-center py-10 mt-5 rounded shadow-lg bg-base-700 color-item">

<LivePreview />

</div>
  
  <LiveError />
</LiveProvider>

```typescript copy
import { AssetsProvider, Deposit, modal } from "@orderly.network/react";

const { dialog } = modal;

export const App = () => {
  return (
    <Button
      onClick={() => {
        dialog({
          title: "Deposit",
          content: <Deposit />,
        });
      }}
    >
      Open deposit dialog
    </Button>
  );
};
```

## APIs

### onOk

- Type: (result)=>void
- Required: false

Callback function after the deposit is complete, providing the status of the deposit transaction.
