# components

## Overview

UI components for positions, position history, liquidation, funding fee history, close position, close all positions, and reverse position.

## Subdirectories

| Directory | Description |
| --------- | ----------- |
| [positions](./positions/index.md) | Desktop/mobile/combine positions tables and row logic |
| [positionHistory](./positionHistory/index.md) | Position history list (desktop + mobile) |
| [liquidation](./liquidation/index.md) | Liquidation history |
| [fundingFeeHistory](./fundingFeeHistory/index.md) | Funding fee history UI and button |
| [positions/closePosition](./positions/closePosition/index.md) | Single position close (desktop + mobile) |
| [closeAllPositions](./closeAllPositions/index.md) | Close all positions widget |
| [reversePosition](./reversePosition/index.md) | Reverse position dialog/widget |
| [shared](./shared/index.md) | Shared hooks (e.g. useTabSort) |
| [rwaStatus](./rwaStatus/index.md) | RWA status tag |
