# closePosition

## Overview

Close single position: desktop (inline inputs) and mobile (slider/dialog). Renders `ClosePositionWidget` which uses `useClosePositionScript` and switches layout by `useScreen()`.

## Files

| File | Description |
| ---- | ----------- |
| [closePosition.widget](./closePosition.widget.md) | Widget + desktop/mobile switch |
| [closePosition.ui](./closePosition.ui.md) | Desktop close UI |
| [closePositions.mobile.ui](./closePositions.mobile.ui.md) | Mobile close UI |
| [closePosition.script](./closePosition.script.md) | Close order state and submit |
| [components/confirmDialog](./components/confirmDialog.md) | Market close confirm dialog |
| [components/quantitySlider](./components/quantitySlider.md) | Mobile quantity slider |
