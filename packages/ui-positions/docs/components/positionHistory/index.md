# positionHistory

## Overview

Position history list (desktop table and mobile list). Exposes `PositionHistoryWidget` and `MobilePositionHistoryWidget`.

| File | Description |
| ---- | ----------- |
| [positionHistory.widget](./positionHistory.widget.md) | Widgets that use `usePositionHistoryScript` and render desktop/mobile UI |
| [positionHistory.ui](./positionHistory.ui.md) | `PositionHistory`, `MobilePositionHistory` presentational components |
| [positionHistory.script](./positionHistory.script.md) | Data and filter state for history |
| [desktop/usePositionHistoryColumn](./desktop/usePositionHistoryColumn.md) | Desktop column config |
| [mobile/](./mobile/index.md) | Mobile cell and items |
