# i18n.ts

## i18n.ts responsibility

Creates and configures the single i18next instance used by the package: default resources (English), language detector (localStorage and cookie), fallback language, and interpolation (escapeValue: false). Does not register React bindings; those are provided by I18nextProvider in provider.tsx.

## i18n.ts exports

| Name | Type | Role | Description |
|------|------|------|-------------|
| resources | object | Initial bundles | { [defaultLng]: { [defaultNS]: en } } |
| i18n | default | i18next instance | createInstance().use(languageDetector); init() |

## i18n instance configuration

| Option | Value | Description |
|--------|-------|-------------|
| fallbackLng | defaultLng | Fallback locale |
| interpolation.escapeValue | false | No XSS escape (React safe) |
| resources | { en: { translation: en } } | Default namespace bundle |

## Language detector

- **Storage**: lookupLocalStorage = i18nLocalStorageKey, lookupCookie = i18nCookieKey.
- **Caches**: ["localStorage", "cookie"].

## i18n.ts dependency

- **Upstream**: constant (defaultLng, defaultNS, i18nLocalStorageKey, i18nCookieKey), locale/en.
- **Downstream**: useTranslation, useLocaleCode, provider, backend, utils.

## i18n.ts Example

```typescript
import i18n from "@orderly.network/i18n";

await i18n.changeLanguage("zh");
const lang = i18n.language;
i18n.on("languageChanged", (lng) => console.log(lng));
```
