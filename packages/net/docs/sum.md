# sum.ts

## Overview

Simple utility function that adds two numbers. Primarily used for testing or as a minimal example export.

## Exports

### `sum`

| Property | Type | Description |
| -------- | ---- | ----------- |
| Signature | `(a: number, b: number) => number` | Returns `a + b` |

## Usage example

```typescript
import { sum } from "@orderly.network/net";

sum(1, 2); // 3
```
