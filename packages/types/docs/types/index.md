# types

## Overview

This directory contains the main API and WebSocket type definitions used across the Orderly ecosystem. The `API` namespace covers REST response shapes; `WSMessage` covers WebSocket message payloads.

## Files

| File | Language | Description |
|------|----------|-------------|
| [api](./api.md) | TypeScript | `API` namespace (tokens, ChainDetail, MarketInfo, Symbol, Order, AlgoOrder, Position, FundingRate, AccountInfo, AssetHistory, Referral, etc.) and `WSMessage` namespace (Ticker, MarkPrice, Position, Order, AlgoOrder, Holding, Announcement) |
