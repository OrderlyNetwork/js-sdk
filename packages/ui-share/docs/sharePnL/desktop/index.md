# sharePnL/desktop

## Overview

Desktop Share PnL: single poster preview, format/options selectors, optional message with checkbox, background carousel, and bottom copy/download buttons.

## Files

| File | Language | Description |
|------|----------|-------------|
| [content](./content.md) | TSX | `DesktopSharePnLContent` (main container). |
| [pnlFormat](./pnlFormat.md) | TSX | `PnlFormatView` (roi_pnl / roi / pnl radio). |
| [checkbox](./checkbox.md) | TSX | `Checkbox` (controlled, with custom SVG). |
| [buttons](./buttons.md) | TSX | `PrevButton`, `NextButton` for carousel. |
| [bottomBtns](./bottomBtns.md) | TSX | `BottomButtons` (Download, Copy). |
| [options](./options.md) | TSX | `ShareOption` (optional field checkbox + label). |
| [message](./message.md) | TSX | `Message` (checkbox + input for custom message). |
| [carousel](./carousel.md) | TSX | `CarouselBackgroundImage` (Embla, prev/next, thumbnails). |
