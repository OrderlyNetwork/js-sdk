# carousel/index

## Overview

Re-exports the carousel as a compound component: Carousel with Content, Item, Next, Previous, and indentify (identifier) as static properties.

## Exports

- **Carousel**: Base component with `Carousel.Content`, `Carousel.Item`, `Carousel.Next`, `Carousel.Previous`, `Carousel.indentify`.

## Usage example

```tsx
import { Carousel } from "./carousel";
<Carousel>
  <Carousel.Content>...</Carousel.Content>
  <Carousel.indentify />
</Carousel>
```
