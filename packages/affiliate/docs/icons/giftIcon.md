# icons/giftIcon.tsx

## Responsibility of GiftIcon

SVG gift icon component. Accepts IconProps (size, className, SVG props). Used in affiliate/referral UI for rewards or gifts.

## GiftIcon Props

Extends IconProps (size?, className?, SVGProps<SVGSVGElement>).

## GiftIcon Example

```tsx
<GiftIcon size={24} className="text-primary" />
```
