# install.tsx

## Overview

Side-effect module that can register the affiliate layout as an extension (currently the extension registration is commented out). Imports layout components and `layoutBuilder` from the layout module.

## Exports

- No public exports; file is imported for side effects (e.g. from `src/index.ts` via `./install`).

## Usage Example

```ts
// Typically imported by index.ts
import "./install";
```
