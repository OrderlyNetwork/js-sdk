# traderIcon.tsx

## Overview

SVG icon component for the trader role. Uses `IconProps` from utils/types (size, className, SVG props).

## Props

| Name | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| size | number | No | | Width/height |
| className | string | No | | CSS class |
| ...rest | SVGProps | No | | SVG element props |

## Usage Example

```tsx
import { TraderIcon } from "../components/traderIcon";
<TraderIcon size={18} />
```
