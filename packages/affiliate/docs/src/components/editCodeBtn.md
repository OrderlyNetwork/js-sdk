# editCodeBtn.tsx

## Overview

Button that wraps EditIcon for editing referral code. Props: optional size and onClick.

## Props

| Name | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| size | number | No | | Icon size |
| onClick | () => void | No | | Click handler |

## Usage Example

```tsx
import { EditCode } from "./editCodeBtn";
<EditCode onClick={openEditModal} />
```
