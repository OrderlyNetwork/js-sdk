# utils

## Overview

Shared utilities: types (e.g. `DateRange`, `IconProps`), referral link and copy helpers, date formatting, SWR key generator, decimal formatting, chart data helpers, and mock referral data.

## Files

| File | Language | Description |
|------|----------|-------------|
| [types.ts](types.md) | TypeScript | Shared types: DateRange, SummaryFilter, BarDayFilter, IconProps |
| [utils.ts](utils.md) | TypeScript | Referral link, copy, date parse/format, generateData |
| [swr.ts](swr.md) | TypeScript | generateKeyFun for paginated SWR keys with date range |
| [decimal.ts](decimal.md) | TypeScript | refCommify for formatted dollar values |
| [chartUtils.ts](chartUtils.md) | TypeScript | fillData for volume chart data |
| [mockData.ts](mockData.md) | TypeScript | MockData.referralInfo for development |
