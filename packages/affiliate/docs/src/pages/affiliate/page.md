# page.tsx (affiliate)

## Overview

Exports the affiliate dashboard page component (AffiliatePage). Composes summary, referral link, referral codes, commission and referees, etc.

## Exports

| Export | Description |
|--------|-------------|
| `AffiliatePage` | Affiliate dashboard page |

## Usage Example

```tsx
import { AffiliatePage } from "./page";
<AffiliatePage />
```
