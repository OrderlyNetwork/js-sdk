# page.tsx (dashboard)

## Overview

Exports the main dashboard page component that shows tab (affiliate/trader) and content. index.tsx re-exports DashboardPage, HomePage, AffiliatePage, TraderPage.

## Exports

| Export | Description |
|--------|-------------|
| `DashboardPage` | Main dashboard page (from page.tsx) |
| `HomePage` | From ../home |
| `AffiliatePage` | From ../affiliate |
| `TraderPage` | From ../trader |

## Usage Example

```tsx
import { DashboardPage, AffiliatePage, TraderPage } from "./index";
```
