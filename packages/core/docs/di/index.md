# di (Dependency Injection)

## Overview

Lightweight dependency injection: global container and `SimpleDI` static API for registering and resolving services by name.

## Files

| File | Language | Summary |
| ---- | -------- | ------- |
| [container.ts](./container.md) | TypeScript | `Container` class: register by class or by name, get, getAll. |
| [simpleDI.ts](./simpleDI.md) | TypeScript | `SimpleDI`: register, registerByName, get, getOr, getAll; uses global container. |
