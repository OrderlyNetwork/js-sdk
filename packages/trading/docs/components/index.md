# components

UI components for the trading package: shared base components, desktop layout and panels, and mobile-specific components.

## Subdirectories

| Directory | Description |
|-----------|-------------|
| [base/](base/index.md) | Shared order book, last trades, position header, countdown |
| [desktop/](desktop/index.md) | Desktop order book, data list, asset view, risk rate, layout, order book & trades, notify |
| [mobile/](mobile/index.md) | Mobile order book, order book & entry, data list, bottom nav, account sheet, portfolio sheet, funding rate, tradingview, top tab, tips, trade data, last trades |
