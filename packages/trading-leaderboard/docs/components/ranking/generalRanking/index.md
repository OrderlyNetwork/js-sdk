# generalRanking

General ranking list: fetches and displays ranking by volume or PnL with date range and address search.

|------|----------|-------------|

| `generalRanking.widget.tsx` | TSX | Widget component |
| `generalRanking.script.ts` | TypeScript | useGeneralRankingScript (data fetching, pagination) |
