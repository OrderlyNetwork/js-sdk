# sidebar.widget

> Location: `packages/ui-scaffold/src/components/sidebar/sidebar.widget.tsx`

## Overview

Widget that uses `useSideNavBuilder(props)` and renders `SideBar` with the resulting state. Accepts `Partial<SideBarProps>`.

## Usage example

```tsx
import { SideNavbarWidget } from "@orderly.network/ui-scaffold";

<SideNavbarWidget menus={[...]} />
```
