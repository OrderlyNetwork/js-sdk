# scaffold

> Location: `packages/ui-scaffold/src/components/scaffold`

## Overview

Main layout component: provides `Scaffold` (desktop/mobile), `ScaffoldProvider`, `ScaffoldContext`, and script/UI split. Renders either desktop (sidebar + content + footer) or mobile layout and supplies router, expand state, chain support check, and announcement state to children.
