# announcementCenter

## Overview

Announcement center panel: title, scrollable list of announcement items with expand/collapse, and icons used in headers.

| [icons.md](icons.md) | TSX | SVG icons: Battle, Campaign, ArrowRightShort, Fund, Announcement, Security |
