# components

## Overview

React components for the vaults experience: all-vaults list (grid/list), vault card, deposit/withdraw flows, header, introduction, and provider that wires API/store.

## Subdirectories

| Directory | Description |
|-----------|-------------|
| [all-vaults](./all-vaults/index.md) | All vaults list, view mode toggle, desktop/mobile layouts |
| [vault-card](./vault-card/index.md) | Single vault card widget, script, UI, constants |
| [vault-operation](./vault-operation/index.md) | Deposit/withdraw forms, deposit+withdraw dialog/sheet, latest deposit/withdraw |
| [vaults-header](./vaults-header/index.md) | Header widget, desktop/mobile, script |
| [vaults-introduction](./vaults-introduction/index.md) | Introduction widget and script |
| [provider](./provider/index.md) | VaultsProvider: config, fetch vault list and overall info on mount |
