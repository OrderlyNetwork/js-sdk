# vaults-header

## Overview

Vaults page header: desktop and mobile variants driven by useVaultsHeaderScript. Renders header content (e.g. title, stats, custom headerImage from config).

## Files

| File | Language | Description |
|------|----------|-------------|
| [vaults-header.widget](./vaults-header-widget.md) | TSX | VaultsHeaderWidget: useScreen + useVaultsHeaderScript; renders desktop or mobile |
| [vaults-header.desktop](./vaults-header-desktop.md) | TSX | Desktop header layout |
| [vaults-header.mobile](./vaults-header-mobile.md) | TSX | Mobile header layout |
| [vaults-header.script](./vaults-header-script.md) | TypeScript | useVaultsHeaderScript: data for header (e.g. overall info, config) |
| [index](./vaults-header-index.md) | TypeScript | Re-exports widget, desktop, script |
