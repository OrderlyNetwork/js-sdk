# version.ts

> Location: `packages/ui-transfer/src/version.ts`

## Overview

Exposes the package version string and attaches it to `window.__ORDERLY_VERSION__["@orderly.network/ui-transfer"]` for runtime version reporting (browser only).

## Exports

- **Default export:** version string (e.g. `"2.9.1"`).

## Usage example

```ts
import version from "@orderly.network/ui-transfer/version";
// or
// window.__ORDERLY_VERSION__["@orderly.network/ui-transfer"]
```
