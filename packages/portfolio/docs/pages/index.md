# pages

## Overview

Page modules for portfolio: overview (dashboard, performance, assets, history, funding, distribution, transfers, vaults), assets (list + convert), fee tier, API key management, setting, positions, orders, and history.

## Subdirectories

| Directory | Description |
|-----------|-------------|
| [api/](api/index.md) | API key manager UI, dialogs (create, edit, delete, created). |
| [assets/](assets/index.md) | Assets page, assets list (desktop/mobile), convert page. |
| [feeTier/](feeTier/index.md) | Fee tier table, data source, columns, icons. |
| [history/](history/index.md) | History page and index. |
| [orders/](orders/index.md) | Orders page. |
| [overview/](overview/index.md) | Overview dashboard, performance, assets chart, asset history, funding, distribution, transfer/vault history, mobile widgets. |
| [positions/](positions/index.md) | Positions page. |
| [setting/](setting/index.md) | Setting page, widget, desktop/mobile UI, script. |
