# shared

Shared column helpers and hooks used by multiple market list components.

## Files

| File | Description |
|------|-------------|
| `column.tsx` | `getSymbolColumn`, `getLastColumn`, `get24hPercentageColumn` – column definitions for symbol, last price, 24h change |
| `hooks/useTabSort.ts` | `useTabSort({ storageKey })` – persist and restore tab sort in storage |
| `hooks/useFavoritesExtraProps.tsx` | `useFavoritesProps()` – extra props for favorites list |
