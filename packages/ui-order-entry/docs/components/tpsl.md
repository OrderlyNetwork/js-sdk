# tpsl.tsx (OrderTPSL)

## Overview

TP/SL section: switch to enable TP/SL, position type (Full/Partial), TP and SL trigger/pnl/offset/ROI inputs, reduce-only switch, and button to open advanced TPSL sheet. Uses order entry context for setOrderValue and errors.

## Props

Includes `quote_dp`, `switchState`, `onSwitchChanged`, `orderType`, `errors`, `setOrderValue`, `reduceOnlyChecked`, `onReduceOnlyChange`, `values` (position_type, tp, sl), `showTPSLAdvanced`, `onChange`.
