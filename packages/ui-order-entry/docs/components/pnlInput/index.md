# pnlInput

## Overview

PnL-based input for TP/SL: provider, context, widget, UI, and builder script. Used in TPSL advanced flow.

## Files

| File | Description |
|------|-------------|
| pnlInputProvider.tsx | Provider for PnL input state |
| pnlInputContext.tsx | Context and hook |
| pnlInput.widget.tsx | Widget wrapper |
| pnlInput.ui.tsx | UI component |
| useBuilder.script.ts | Builder logic |
