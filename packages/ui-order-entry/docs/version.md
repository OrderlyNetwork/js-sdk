# version.ts

## Overview

Registers the package version on `window.__ORDERLY_VERSION__["@orderly.network/ui-order-entry"]` for runtime version reporting. Exports the version string as default.

## Exports

- **Default:** `"2.9.1"` (string)

## Usage

```ts
import version from "@orderly.network/ui-order-entry/version";
// or read from window.__ORDERLY_VERSION__
```
