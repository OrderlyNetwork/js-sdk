# common — Shared chart primitives

Reusable axis, legend, and grid components used by custom D3-based charts (not by Recharts-based orderly charts).

## Files

| File | Language | Description | Link |
|------|----------|-------------|------|
| axis.tsx | React/TSX | D3 axis (bottom/left) with ChartContext; draws ticks and removes domain | [axis.md](axis.md) |
| legend.tsx | React/TSX | Simple legend group with optional children or default "Legend" text | [legend.md](legend.md) |
| grid.tsx | React/TSX | Placeholder Grid component (no visual output) | [grid.md](grid.md) |
