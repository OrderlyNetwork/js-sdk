# icons

## Overview

SVG icon components used across the wallet connector UI: MoreIcon, DisconnectIcon, CloseIcon, ArrowRightIcon, ArrowRightLinearGradientIcon, CloseSquareFillIcon.

## Exports

| Component | Description |
|-----------|-------------|
| **MoreIcon** | Three-dots “more” icon. |
| **DisconnectIcon** | Logout/disconnect icon. |
| **CloseIcon** | X / close icon. |
| **ArrowRightIcon** | Right arrow. |
| **ArrowRightLinearGradientIcon** | Right arrow with gradient. |
| **CloseSquareFillIcon** | Filled close square. |

All accept standard SVG/React props (e.g. `className`, `size` where applicable).
