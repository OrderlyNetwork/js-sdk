# AddAbstractWallet

## Overview

**AddAbstractWallet** is an “Add Abstract wallet” row used in **RenderNonPrivyWallet**. It shows a single Abstract option and calls `connect({ walletType: WalletConnectType.ABSTRACT })`. Auto-opens when `targetWalletType === WalletType.ABSTRACT` and auto-closes after a short delay.

## Exports

- **AddAbstractWallet** – Function component; no props.
