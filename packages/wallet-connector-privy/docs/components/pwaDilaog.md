# PwaDialog

## Overview

**PwaDialog** is a modal (created via `modal.create`) that explains how to install the app as PWA: it shows the current domain and step-by-step instructions (e.g. share/add to home). Used from ConnectDrawer on mobile.

## Exports

- **PwaDialog** – Modal component (use via `modal.show(PwaDialog)`).

## Usage example

```tsx
modal.show(PwaDialog);
```
