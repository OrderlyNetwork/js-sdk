# hooks

## Overview

Custom hooks for wallet connection and chain switching. **useWallet** is the main hook that unifies Privy, Wagmi, Solana, and Abstract state and exposes `connect`, `disconnect`, `setChain`, `switchWallet`, and `onDisconnect`.
