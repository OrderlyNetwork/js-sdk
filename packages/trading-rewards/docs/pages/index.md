# pages

## Overview

Root of the Trading Rewards page: barrel exports and the main home page component. The `index` subfolder contains the home page composition, provider, and feature widgets (title, curEpoch, availableToClaim, stakeBooster, rewardHistory).

## Subdirectories

| Directory | Description |
|-----------|-------------|
| [index/](index/index.md) | Home page: provider, page, title, curEpoch, availableToClaim, stakeBooster, rewardHistory. |

## Barrel (pages/index)

The `index` folder exports: `Title`, `TitleWidget`, `TitleConfig`, `CurEpochWidget`, `CurEpoch`, `useCurEpochScript`, `AvailableToClaimWidget`, `AvailableToClaim`, `StakeBoosterWidget`, `StakeBooster`, `RewardsHistoryWidget`, `TradingRewardsProvider`, `HomePage`.
