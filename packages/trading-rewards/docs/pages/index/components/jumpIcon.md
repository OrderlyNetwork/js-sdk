# jumpIcon

## Overview

Small arrow-in-box SVG icon indicating an external or “jump” action. Used for “Claim” and “Stake” links.

## Exports

### JumpIcon (component)

No props. Renders 16×16 SVG with currentColor fill.

## Usage example

```tsx
<Flex onClick={goToClaim}>
  <Text>Claim</Text>
  <JumpIcon />
</Flex>
```
