# esOrderlyIcon

## Overview

esORDER token icon: circular gradient (white to light gray) with inner gradient symbol. Used alongside OrderlyIcon where both ORDER and esORDER are shown.

## Exports

### EsOrderlyIcon (component)

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| className | string | No | SVG class. |

## Usage example

```tsx
<EsOrderlyIcon className="oui-w-5 oui-h-5" />
```
