# title

## Overview

Page title block: heading, divider, subtitle, and “Learn more” doc link. Config comes from `TradingRewardsContext` via `useTitleScript`.

## Files

| File | Language | Description |
|------|----------|-------------|
| [title.script](title.script.md) | TSX | `TitleConfig` type and `useTitleScript`. |
| [title.ui](title.ui.md) | TSX | `Title` and `MultiLineText` (doc link). |
| [title.widget](title.widget.md) | TSX | `TitleWidget` that wires script to UI. |
