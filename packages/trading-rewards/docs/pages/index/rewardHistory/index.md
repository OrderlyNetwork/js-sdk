# rewardHistory

## Overview

Reward history list: combines epoch list with wallet rewards and claimed state, supports desktop (DataTable) and mobile (ListView), with pagination and optional per-row rewards tooltip.

## Files

| File | Language | Description |
|------|----------|-------------|
| [rewardsHistory.script](rewardsHistory.script.md) | TSX | `useRewardsHistoryScript`, `ListType`, `RewardsHistoryReturns`. |
| [rewardsHistory.ui](rewardsHistory.ui.md) | TSX | `RewardHistory`, desktop/mobile list, and timestamp formatting. |
| [rewardsHistory.widget](rewardsHistory.widget.md) | TSX | `RewardsHistoryWidget` that wires script to UI. |
