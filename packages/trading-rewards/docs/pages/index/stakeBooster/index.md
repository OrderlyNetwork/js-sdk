# stakeBooster

## Overview

Stake booster block: average staked amount, booster multiplier, and “Stake” link that opens the app staking page. Booster value is derived from `est_stake_boost`.

## Files

| File | Language | Description |
|------|----------|-------------|
| [stakeBooster.script](stakeBooster.script.md) | TSX | `useStakeBoosterScript`, stakeNow URL, and booster calculation. |
| [stakeBooster.ui](stakeBooster.ui.md) | TSX | `StakeBooster` UI with two stat cards. |
| [stakeBooster.widget](stakeBooster.widget.md) | TSX | `StakeBoosterWidget` that wires script to UI. |
